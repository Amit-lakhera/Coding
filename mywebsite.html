INTRODUCTION TO HTML


What is an HTML?


  • An HTML stands for "HYPERTEXT MARKUP LANGUAGE".

  • HTML is used to create web pages and web applications.

  • HTML is widely used web language on the web.

  • We can create a static website by HTML only.

  • Technically,HTML is a markup language rather than a programming language.

State the Basic HTML tags.


  1. !DOCTYPE html tag
    2. It defines the document type or it instructs the browser about the version of the browser.


  2. HTML tag
    3. This tag informs the browser that it is an HTML document text.It is a container for all other elements of html.


  3. HEAD tag
    4. It should be the first element inside the html tag which contains the metadata.It must be closed before the body tag opens.


  4. TITLE tag
    5. It is used to add title of that html page which appears at the top the window's browser.It must be placed inside the head tag and should close immediately.


  5. BODY tag
    6. Text between body tag, describes the body content of the page that is visible to the end user. This tag contains the main content of the html document.


  6. HEADING tag
    7. Heading tag is used to mark headings in HTML. From h1 to h6, we have tags for the most important to least important heading.


  7. PARAGRAPH tag
    8. It is use to describe the paragraph of the web page in HTML.
